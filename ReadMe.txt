!!!Chỉ dùng được với youtube
Bước 1: Truy cập Google Coolab
Bước 2: Tìm tiện ích Cookie-Editor và thêm vào googlechome (hay brower khác cũng được)
Bước 3: Vào file -> upload notebook -> chọn file DownYoutubeVideoProV1.pynb
Bước 4: Vào trang Youtube -> mở video muốn tải / playlist 
Bước 5: Mở Cookie-Editor -> nhấn dấu "-->" ( Export) để xuất cookies -> chọn Nestcape
Bước 6: Trong thư mục Google Coolab -> tạo file cc.txt -> dán cookies vừa lấy được vào
Bước 6: Chạy từng phần code, thay link video có sẵn bằng link muốn tải.
